# Welcome to Jobfynder Research Vault

This vault is synced via Git. Notes are auto-generated by Hermes after research tasks.

## Structure
- `research/` — Research notes by topic
- `daily/` — Daily summaries and session notes
- `projects/` — Active project context

## How It Works
1. Hermes runs research → saves `.md` notes here
2. Git auto-commits and pushes
3. Pull on laptop → open in Obsidian
4. Edit on laptop → push back → Hermes reads updates
