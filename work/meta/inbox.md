# Idea Inbox — Quick Capture

This is the vault's inbox for rough ideas, links, and things to review later.
Nothing here needs to be built today. It's reviewed periodically.

## How to Use
When you come across something interesting (idea, YouTube video, blog, article, thought):
- Add a line to this file
- That's it. One line is enough.
- Periodically, review the inbox — promote ideas to `product/` or `research/`, delete the rest.
